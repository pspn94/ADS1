1. Criar nova pasta pelo terminal do VS Code
2. Usar 'git clone' + url para clonar no PC o projeto git
3. Usar o 'code .' para abrir o projeto do VS
4. Criar branch com botão no lado esquerdo inferior do VS Code
Creat branch from main > colocar nome da branch
5. Fazer alterações devidas
6. Usar o Source Control para visualizar alterações.
7. Usar o botão '+' para mudar de Changes para Stage
8. Escrever mensagem e Clicar no Botão Commit para commit the changes.
9. Sync changes para subir para o Git
10. Utilizar o botão da núvem no lado esquerdo inferior para publicar branches
